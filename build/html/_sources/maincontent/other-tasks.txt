In addition to Search, Dataset Explorer, and Gene Signature/List, the
tranSMART toolbar includes the following dropdown menu and tool tabs:

-  **Utilities**

   Options under this menu include Help (takes you to the online help
   manual), Contact Us (opens email for questions, problem reports, or
   any other feedback about the tranSMART application), About (version
   information for tranSMART application), and access to Login and
   Logout

-  **Admin**

Perform user administration functions such as creating users, assigning
roles to users, and enabling or disabling user accounts.

This tool tab is visible only to users who are assigned the role
ROLE\_ADMIN.
